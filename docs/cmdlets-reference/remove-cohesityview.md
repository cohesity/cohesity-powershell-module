# Remove-CohesityView

## SYNOPSIS
Removes a Cohesity View.

## SYNTAX

```
Remove-CohesityView -Name <string> [<CommonParameters>]
```

## DESCRIPTION
Returns success if the view is deleted.

## EXAMPLES

### EXAMPLE 1
```
Remove-CohesityView -Name "Test-View"
```

Removes a view with the name "Test-View".

## PARAMETERS

### -Name
