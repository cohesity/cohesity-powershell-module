# Get-CohesityViewAllowlist

## SYNOPSIS
Get allowlist IP(s) for a given view.

## SYNTAX

```
Get-CohesityViewAllowlist [-ViewName] <String> [<CommonParameters>]
```

## DESCRIPTION
Get allowlist IP(s) for a given view.

## EXAMPLES

### EXAMPLE 1
```
Get-CohesityViewAllowlist -ViewName view1
```

Get the allowlist for view1.
